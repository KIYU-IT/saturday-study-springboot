💡 사용 팁

  • 먼저 "서버 상태 확인"을 클릭하여 API 서버가 실행 중인지 확인하세요
  • "샘플 주문 생성"을 클릭하여 주문을 만드세요
  • 응답에서 주문 ID를 복사하여 위의 입력창에 붙여넣으세요
  • "다음 단계 진행"을 여러 번 클릭하여 상태 변화를 확인하세요
  • 각 상태에서 "주문 취소"를 시도해보며 제약사항을 확인하세요
  • "가능한 액션"을 확인하여 현재 상태에서 할 수 있는 일들을 파악하세요
